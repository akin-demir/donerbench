# Media assets

- **`demo.mp4`** — a screen recording of a benchmark run (referenced from the root `README.md`). Don't commit it as a plain blob: prefer a GitHub attachment/release for the README, or Git LFS (`git lfs install`; `assets/*.mp4` is already tracked in [`../.gitattributes`](../.gitattributes)) if you want it versioned in the repo.
- The app logo lives at [`../frontend/public/logo.png`](../frontend/public/logo.png) and is also used as the web-app favicon. Replace that file to change both the README logo and the browser icon.
